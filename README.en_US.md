# How to run this project

## 1) Install Node.js(Necessary)
 - Visit webpage: https://nodejs.org and download the latest version of Node.js install package.
 - Follow the instruction to install Node.js on your computer, and turn on your Terminal and input:

  ```bash
  node -v  # it will be OK if it return any version.
  ```

  ## 2) Run

  In your Terminal, input:
  ```bash
  node app.mjs
  ```
  then you can visite `http://localhost:9527` on your Browser.
